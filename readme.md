# Forschungsorientierte Einführung in die Experimentelle Philosophie

This is a repository for the course “Forschungsorientierte Einführung in die Experimentelle Philosophie” (research-oriented introduction to experimental philosophy), organized by Stephan Kornmesser and me in the winter term of 2025/2026 at the University of Oldenburg.
